GitHub's new drag and drop feature.

Its now much easier to upload files to your repository.

Git worktree

It is now possible to upload files via a new drap and drop feature on GitHub. Simply open GitHub in your browser, select the 'Upload files' button and drag in a file. This will appeal to those who do not like to go near the command line.
Many developers stay away from Git because of its command line surroundings. This feature was added this Febuary and has been welcomed by all.

Initially files could only be uploaded via the command line version or the GitHub desktop application. Now it can be done like so.

Does this mean the end of the git push command. I think not. True hardcore users of Git will probably be reluctant to use this feature, and that is possible by simply ignoring its excistence and using the original technique.
Only time will tell if this type of file upload will prove to be successful in the long run. We will keep you posted.
