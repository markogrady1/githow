About Us

What is the aim of GitHow.

This site was designed and created by Mark O Grady. The look and feel of the site was inspired by the folks at HTML5 up.

GitHow provides news and information with regards to Git and GitHub. It also provides the all important commands and best practices. GitHow also provides a list of the hottest repositories

The history page provides an overview of the history of Git is provided. Also, information about contributing to Git is provided. It also contains links to the IRC channel and the email address for those individual who may be interested in making Git a better tool than it already is.

GitHow also provides articles about the latest news and events in the world of Git and GitHub.
